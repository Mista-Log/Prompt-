Three AI Chatbot Categories

DeFi (Sidekick)

    Function: Helps users navigate Solana DeFi (swaps, yields, lending). Features:
    • Real-time APY comparisons (e.g., Kamino vs. Marginfi).
    • Gasless intent-based swaps (e.g., “Swap 10 SOL to highest-yielding stablecoin”).
    • Solana Advantage: Uses Solana’s speed/low fees to execute trades in seconds.

NFT (Navigator)

    Function: Guides users through Solana’s NFT ecosystem. Features:
    • Rarity analysis (Tensor or Magic Eden API).
    • Trend alerts (e.g., “Top 3 rising NFT collections this hour”).
    • Solana Advantage: Leverages compression for instant NFT data queries.


Devs (Builder)

    Function: Assists developers building on Solana. Features:
    • Anchor framework code snippets.
    • Debugging help (e.g., “Fix this CPI error”).
    • Solana Advantage: Direct access to Solana devnet/testnet for live testing.
